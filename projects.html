VetAlign — Medical Record System for Veterinary Clinics

A full-stack medical record and practice management system designed specifically for Danish veterinary clinics. VetAlign enables clinics to digitise patient records, manage appointments, and streamline administrative workflows in a secure, GDPR-compliant environment.

  • Designed and implemented the full system architecture — from database schema to frontend interface — as a solo developer.
  • Built a responsive web application using TypeScript (React) on the frontend and C# on the backend, with a SQL database layer.
  • Developed role-based access control, audit logging, and data encryption to meet healthcare compliance requirements.
  • Deployed and maintained the system in a cloud environment with CI/CD pipelines and automated testing.
  • Conducted user interviews with veterinary professionals to iteratively refine the product based on real clinical workflows.
Full-Stack C# TypeScript React SQL CI/CD Cloud
